Functional Applications

With Spring Cloud Stream 3.x adding functional support, users can build Source, Sink and Processor applications by merely implementing the Java Util's Supplier, Consumer and Function interfaces respectively.

Spring Cloud Stream also has the function composition support which we show in the function composition feature guide.
